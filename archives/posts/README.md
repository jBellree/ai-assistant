# Posts Archive

Every piece of social content generated through the social-content skill is saved here.

## Purpose

FCA record-keeping. As a regulated firm (FRN: 649675), DSG Financial Services Ltd needs to be able to evidence its marketing communications. This folder is that evidence.

## File naming

`YYYY-MM-DD-[pillar]-[brief-slug].md`

Examples:
- `2026-04-16-deal-spotlight-porsche-taycan.md`
- `2026-04-14-rate-watch-northridge-drop.md`
- `2026-04-11-underwriting-lhd-vehicles.md`

## What each file contains

- Date, pillar, platforms, image template used
- Full post copy (exactly as posted)
- Image variables passed to hcti.io (older posts reference Bannerbear, which was the previous image system)
- Image URL (for reference)

## Do not delete files from this folder

These are compliance records. Archive, don't delete.
