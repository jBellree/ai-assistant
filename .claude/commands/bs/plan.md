Invoke the `daily-planning` skill.

- **Default (no args):** run the full morning check-in — Phases 0 through 6 — and write today's `sessions/YYYY-MM-DD.md`.
- **`--close`:** skip the morning phases. Jump straight to the end-of-day close-out on today's session file (recap each block, batch-tick Reminders, log decisions, carry items forward).

Follow the rules defined in the skill: direct, no fluff, under 5 minutes of B's time, preview → confirm → commit for every Airtable and Reminders write.
