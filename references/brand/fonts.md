# Magnitude Finance — Fonts (from official brand guidelines)

## Heading Font

**Poppins Semi-Bold or Bold** (Google Fonts — free)
- Used for all headings and titles
- Title focus word(s) in Burnt Orange `#ED792B`
- Keep orange highlight under 50% of total title length
- hcti.io: use `Poppins` as the font name, weight 600 (Semi-Bold) or 700 (Bold)

## Body / Copy Font

**Montserrat** (Google Fonts — free)
- Used for all body copy, captions, contact details, table values
- Never heavier than Medium weight (500)
- Quote marks: Montserrat SemiBold in Burnt Orange

## Sizing Guide (1080×1080 canvas)

| Element | Font | Size | Weight |
|---|---|---|---|
| Main headline | Poppins | 72–90px | Bold 700 |
| Sub-headline | Poppins | 48–60px | Semi-Bold 600 |
| Body / detail text | Montserrat | 28–36px | Regular 400 |
| Contact details | Montserrat | 28px | Regular 400 |
| FCA disclaimer | Montserrat | 18–22px | Regular 400 |
| Table labels | Montserrat | 24px | Medium 500 |
| Table values | Montserrat | 24px | Regular 400 |

## Notes

- Never use Montserrat for headings — that's Poppins only
- Never use Poppins heavier than Bold
- Never use serif fonts anywhere
- Rounded corners on all boxes and cards — this is a brand rule
