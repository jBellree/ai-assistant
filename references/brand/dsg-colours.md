# DSG Group — Brand Colours

## Shared Base Palette

| Name | Hex | Usage |
|---|---|---|
| **Deep Slate** | `#252E3F` | Primary background, email footer |
| **White** | `#FFFFFF` | Email body, text on dark backgrounds |
| **Light Grey** | `#FBFBFB` | Light backgrounds |
| **Teal** | `#29D9D9` | DSG Group / DSG Finance primary accent |
| **Fuchsia** | `#E70175` | DSG Connected primary accent |
| **Azure Blue** | `#017FFF` | DSG Prestige primary accent |
| **Apple Green** | `#0DB729` | CTA buttons |
| **Platinum** | `#C6C6C6` | Supporting / neutral |
| **Burnt Orange** | `#ED792B` | DSG Prestige (secondary) |

## Per-brand accent colour

| Brand | Primary accent | Notes |
|---|---|---|
| DSG Group | Teal `#29D9D9` | H3 headings, bullets, links |
| DSG Finance | Teal `#29D9D9` | |
| DSG Prestige | Azure Blue `#017FFF` | Email buttons/links use Azure Blue, not Teal |
| DSG Vision | Azure Blue `#017FFF` | Same as Prestige |
| DSG Connected | Fuchsia `#E70175` | Higher illustration use (50/50 images vs illustrations) |

## Notes

- Gradients are permitted using palette colours (consistent with DSG website).
- Rounded corners on all text boxes, images (curved rectangles), and cards.
- Email: background `#EFEFEF`, body `#FFFFFF`, footer `#252E3F`.
