# DSG Deal Placement Team Training

A training manual and session pack for DSG's deal placement team, covering how to read UK personal tax returns and company accounts when profiling finance applications for prestige vehicle deals.

## What's in this folder

- **`CLAUDE.md`** — project brief and context for Claude Code sessions. Read this first if you're Claude.
- **`00-tutoring-notes/`** — B's learning notes, written during tutoring sessions.
- **`01-manual/`** — the training manual. The main deliverable.
- **`02-worked-examples/`** — fabricated customer profiles used throughout the training.
- **`03-slide-deck/`** — outline for the live 1-hour session.
- **`04-reference/`** — quick-reference cheat sheets.

## How to pick this up in a new Claude Code session

Start the session with:

```
Read CLAUDE.md and then look at the tutoring notes in 00-tutoring-notes/ to see where we got to. Then continue tutoring me from where we left off.
```

Claude should read the brief, see what's been covered, and pick up without needing a re-explanation.

## Deadline

One week from kickoff. Live session is ~1 hour.

## Audience

5-person deal placement team at DSG, treated as beginners. Training is also designed to be reusable for future new starters.
